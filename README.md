# Вариант №53

## Выражение

$$b \cdot c + \frac{a}{d + e} - \frac{d^2}{b \cdot e} + a \cdot e$$

## Размер обрабатываемых переменных

- $a$ — 32 бит;
- $b$ — 16 бит;
- $c$ — 32 бит;
- $d$ — 16 бит;
- $e$ — 32 бит.

## Наличие знака у обрабатываемых целых чисел

Все обрабатываемые целые числа являются знаковыми.
